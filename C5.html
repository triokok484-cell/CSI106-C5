Computer Organization

1. The Three Subsystems of a Computer

  • A computer consists of 3 subsystems: CPU (Central Processing Unit), Main Memory, and I/O Subsystem.

2. Central Processing Unit (CPU) in Detail

  • Known as the "brain" of the computer, it has 3 main components:
    • ALU (Arithmetic Logic Unit): Performs arithmetic and logical operations.
    • Control Unit: Directs and coordinates the activities of all other components.
    • Registers: High-speed temporary storage areas within the CPU.
      • Program Counter (PC): Always points to the address of the next instruction.
      • Instruction Register (IR): Holds the instruction currently being decoded and executed.

3. Main Memory in Detail

  • Comprises many memory locations, each with a unique address.
  • RAM (Random Access Memory): Read-write memory where data is lost when power is off (volatile). Includes SRAM and DRAM.
  • ROM (Read-Only Memory): Read-only memory where data is not lost when power is off (non-volatile).
  • Cache Memory: Super-fast buffer memory between the CPU and RAM to speed up access.
  • Memory Hierarchy: Arranged by decreasing speed: Registers ‚Üí Cache ‚Üí Main Memory ‚Üí Secondary Storage.

4. Program Execution

  • Machine Cycle: A repeating cycle with 3 core steps:
    1. Fetch ‚Äì Load an instruction from memory.
    2. Decode ‚Äì Interpret the instruction.
    3. Execute ‚Äì Carry out the instruction.
  • CPU and I/O Synchronization: 3 main methods, with Direct Memory Access (DMA) being the most efficient for large data block transfers as it does not require constant CPU intervention.

5. Computer Architectures

  • CISC: Large and complex instruction set.
  • RISC: Small, simple instruction set, which helps in faster execution.
  • Pipelining: Allows for the overlapping execution of multiple instructions to increase throughput.
  • Parallel Processing: Uses multiple processing units to execute work simultaneously.
